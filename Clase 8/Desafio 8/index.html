Recordatorios

  • Siempre revisar el grupo de Whatsapp en caso de que alla noticias de última hora.

  • Nunca poner la diversión antes que la tarea.

  • No descansar hasta que todo el deber este hecho.

  • Siempre tener en cuenta las actividades de los días siguientes para organizarse mejor.

  • Si hay que cancelar una salida para hacer la tarea, se hace.

  • Siempre salir con tiempo extra, tanto para el trabajo como el colegio.
